# speedyPizza_Excel

Eseguita l'elaborazione, in output (nella directory di esecuzione del programma) crea una cartella denominata (ordini_Y-M-D h#m#s) ed all'interno posiziona il file.xlsx(excel) elaborato denominato(ordine_h#m). Se si spunta la checkbox archivia originale, viene creata un'altra cartella denominata( archivioY-M-D) al cui interno viene posizionato il file originale(precedentemente elaborato) denominato(D-hfile.xlsx).
